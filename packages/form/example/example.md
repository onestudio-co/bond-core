Basic usage:

```dart
final loginForm = FormStateNotifier<String, Error>(
fields: {
'email': TextFieldState(null, label: 'Email'),
'password': TextFieldState(null, label: 'Password'),
},
);

// Updating a form field
loginForm.update('email', 'johndoe@gmail.com');

// Submitting the form
loginForm.submit();
```
